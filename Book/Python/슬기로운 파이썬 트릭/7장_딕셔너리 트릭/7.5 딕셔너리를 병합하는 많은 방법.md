## 딕셔너리를 병합하는 많은 방법

```python
xs = {'a': 1, 'b': 2}
ys = {'b': 3, 'c': 4}

zs = {}
zs.update(xs)
zs.update(ys)
```

- 파이썬에서 다중 딕셔너리 병합의 고전적인 해결책은 update메소드를 사용하는 것이다

```python
zs = dict(xs, **ys)
```

- 위와 같이 객체를 푸틑 ** 연산자와 함꼐 dict()내장 함수를 사용하는 방법도 있다
- 하지만 위 방법 모두 두 개의 딕셔너리를 병합하는 경우에만 작동하며 임의의 수의 딕셔너리를 한 번에 결합할 수 없다

```python
zs = {**xs, **ys}
```

- 파이썬 3.5 부터는 ** 연산자가 더 유연해져서 위와 같은 코드를 작성할 수 있따
  - 왼쪽에서 오른쪽 순서로 설정된다
